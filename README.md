#Sistema de Login e CRUD em PHP com MySQL

##Sobre o Projeto

Este projeto foi desenvolvido com base na trilha de estudos do canal Gustavo Neitzke, no YouTube. O objetivo principal foi aprender e aplicar os fundamentos de PHP com MySQL, focando na criação de um sistema completo com operações CRUD

Utilizei o XAMPP como ambiente de desenvolvimento local para realizar a conexão com o banco de dados e testar o sistema, compreendendo na prática:
Conexão entre PHP e MySQL
Criação de banco de dados e tabelas
Utilização de comandos SQL
Inserção, consulta, atualização e exclusão de dados
Estrutura básica de um sistema com autenticação

O projeto não tem foco em arquitetura MVC, programação orientada a objetos ou design avançado, mas sim no entendimento dos conceitos fundamentais de banco de dados e backend

Funcionalidades

O sistema permite: Cadastro de usuários; Login de usuários; Listagem de registros; Pesquisa de usuários; Edição de dados; Exclusão de registros

##Tecnologias Utilizadas:

PHP
MySQL
XAMPP
Bootstrap
HTML
CSS
JavaScript


Este projeto representa meu início no desenvolvimento backend e demonstra minha evolução no entendimento de banco de dados e lógica de sistemas web
